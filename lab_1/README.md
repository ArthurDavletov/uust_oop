# Лабораторная работа 1. Кнопки и формы

Тип приложения: GUI, desktop. Реализация выполнена на Python с PySide6.

## Смысл работы

Цель лабораторной - разобраться с созданием графического интерфейса, стандартными компонентами, свойствами объектов и обработчиками событий. По методичке приложение должно содержать много разнотипных взаимодействующих элементов, меню, реакцию на события мыши и клавиатуры, таймер, изменение свойств в runtime и динамическое создание элементов интерфейса.

## Что проверять на защите

- Понимание понятий объект, свойство, событие и обработчик события.
- Реакции на нажатия мыши, движение мыши, клавиатуру, меню, таймер и изменение размера окна.
- Изменение свойств компонентов до запуска и во время работы приложения.
- Один обработчик для нескольких объектов и взаимодействие одних элементов интерфейса с другими.
- Динамическое создание элементов интерфейса и назначение им обработчиков.

## Запуск

```powershell
pip install -r ../requirements.txt
python main.py
```

Из корня репозитория:

```powershell
python lab_1/main.py
```
