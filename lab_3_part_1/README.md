# Лабораторная работа 3. Часть 1. Круги на форме

Тип приложения: GUI. Реализация выполнена на Python с PySide6.

## Смысл работы

Нужно создать приложение с рабочей областью для рисования, классом круга и собственным контейнером объектов. При клике мышью создаются круги, при перерисовке формы все объекты должны рисовать себя сами, а приложение должно поддерживать изменение размера окна.

## Что проверять на защите

- Создание объекта круга по координатам клика.
- Хранение объектов в отдельном контейнере.
- Отрисовка всех объектов при событии paint.
- Выделение одного или нескольких кругов, в том числе с Ctrl.
- Удаление выделенных объектов через Del.
- Соблюдение инкапсуляции: объект сам проверяет попадание и сам рисует себя.

## Запуск

```powershell
pip install -r ../requirements.txt
python main.py
```

Из корня репозитория:
