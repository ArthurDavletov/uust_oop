# Лабораторная работа 6. Группировка и сохранение

Тип приложения: GUI. Реализация выполнена на Python с PySide6.

## Смысл работы

Работа развивает визуальный редактор: добавляется группировка объектов и сохранение проекта в файл с последующим восстановлением. Группа должна вести себя как единый объект и может содержать другие группы.

## Основные требования

- Группировка и разгруппировка выделенных объектов.
- Реализация групп через паттерн Composite.
- Перемещение группы с сохранением относительного положения дочерних объектов.
- Проверка выхода группы за границы рабочей области.
- Сохранение и загрузка контейнера в человеко-читаемый текстовый файл.
- Восстановление объектов через фабричный подход.
- Использование стандартных диалогов выбора файла.
- Описание нетривиальных пользовательских историй.

Пользовательские истории вынесены в [user_stories.md](user_stories.md).

## Запуск

```powershell
pip install -r ../requirements.txt
python main.py
```

Из корня репозитория:

```powershell
python lab_6/main.py
```
