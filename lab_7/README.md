# Лабораторная работа 7. Дерево объектов, подписка

Тип приложения: GUI. Реализация выполнена на Python с PySide6.

## Смысл работы

Работа развивает редактор из лабораторной 6: добавляется дерево объектов, синхронизация выделения через Observer и новый объект-стрелка, связывающий два объекта или группы.

## Основные требования

- `TreeView` отображает текущее содержимое контейнера, включая вложенные группы.
- Дерево перестраивается при изменениях контейнера по подписке Observer.
- Выделение синхронизируется в обе стороны: из дерева в рабочую область и обратно.
- Стрелка соединяет объект-источник и объект-цель; при перемещении источника цель перемещается вместе с ним.
- Источником и целью могут быть отдельные фигуры или группы.
- Приложение должно корректно вести себя на нетривиальных историях и при циклах связей.

## Пользовательские истории для проверки

1. Создать два объекта, объединить их в группу, создать третью фигуру и проверить синхронизацию выделения группы в дереве и на рабочем поле.
2. Создать две группы, добавить стрелку от первой ко второй, переместить первую группу и убедиться, что вторая перемещается вместе с ней.
3. Создать вложенную группу, сохранить проект, удалить все объекты, загрузить проект и проверить дерево объектов.
4. Создать стрелку между группами, разгруппировать одну из групп и проверить, что дерево и рабочая область остаются согласованными.
5. Создать цикл из стрелок и проверить, что перемещение не приводит к бесконечному каскаду обновлений.
