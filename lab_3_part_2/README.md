# Лабораторная работа 3. Часть 2. MVC

Тип приложения: GUI. Реализация выполнена на Python с PySide6.

## Смысл работы

Приложение работает с тремя целыми числами `A`, `B` и `C` в диапазоне от 0 до 100. Каждое значение отображается и редактируется через несколько компонентов интерфейса, а модель отвечает за проверку ограничений, пересчет и сохранение состояния.

## Основные требования

- Каждое число отображается в трех видах: текстовое поле, числовой ввод и ползунок.
- Изменение одного представления обновляет остальные представления этого же числа.
- Всегда должно выполняться условие `A <= B <= C`.
- Значения сохраняются между запусками приложения.
- Логика значений находится в модели, а не в форме.
- Модель уведомляет представление об изменениях и не рассылает лишние обновления.

## Запуск

```powershell
pip install -r ../requirements.txt
python main.py
```

Из корня репозитория:

```powershell
python lab_3_part_2/main.py
```
