# Лабораторная работа 4. Визуальный редактор

Тип приложения: GUI. Реализация выполнена на Python с PySide6.

## Смысл работы

Лабораторная развивает идеи контейнера и самостоятельных графических объектов из предыдущих работ. Нужно реализовать простой визуальный редактор векторных объектов: создание фигур, выбор, изменение цвета, размера и положения, удаление и контроль выхода за рабочую область.

## Основные требования

- Меню или панель инструментов с доступными типами объектов.
- Добавление графических объектов в рабочую область.
- Собственная иерархия классов фигур.
- Выделение одного или нескольких объектов.
- Манипуляции объектами с клавиатуры.
- Изменение цвета через стандартный диалог.
- Использование собственного контейнера объектов.
- Отделение пользовательского интерфейса от логики классов фигур.

## Запуск

```powershell
pip install -r ../requirements.txt
python main.py
```

Из корня репозитория:
