# Лабораторная работа 5. Жизненный цикл объектов C++ и виртуальность

Тип приложения: консольное. Реализация выполнена на C++20.

## Смысл работы

Работа посвящена виртуальным методам, виртуальному деструктору, безопасному приведению типов, передаче объектов в функции, возврату объектов из функций и влиянию умных указателей на жизненный цикл объектов.

## Что реализовано

- Иерархия классов `Shape`, `ClosedShape`, `Circle`, `Square`, `Segment`.
- Невиртуальные и виртуальные методы с демонстрацией различий при вызове через базовый класс.
- Виртуальный деструктор и отладочный вывод жизненного цикла.
- Методы `classname()` и `isA()` для ручной проверки типа.
- Примеры `static_cast` и `dynamic_cast`.
- Передача объектов по значению, указателю и ссылке.
- Возврат объектов по значению, указателю и ссылке с разбором опасных случаев.
- Использование `unique_ptr` и `shared_ptr`.
