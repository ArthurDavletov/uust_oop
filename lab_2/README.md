# Лабораторная работа 2. Объекты и классы

Тип приложения: консольное. Реализация выполнена на C++20.

## Смысл работы

Лабораторная демонстрирует базовые механизмы ООП в C++: классы, атрибуты и методы, конструкторы и деструкторы, наследование, модификаторы доступа, композицию, статическое и динамическое создание объектов, копирование объектов и указателей.

## Что реализовано

- Классы геометрических объектов: точки, отрезки, фигуры, квадрат, круг.
- Конструкторы без параметров, с параметрами, копирования и перемещения.
- Отладочный вывод в конструкторах, деструкторах и основных методах.
- Наследование и переопределение поведения через базовые классы.
- Композиция через вложенный объект и через указатели.
- Демонстрация копирования объекта, копирования указателя, object slicing и работы через указатель на базовый класс.

## Запуск

Из корня репозитория:

```powershell
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

```powershell
.\build\Release\lab2.exe
```

Linux/macOS:

```bash
./build/lab2
```
